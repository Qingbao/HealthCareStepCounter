HealthCareStepCounter
=====================

Coming soon: rewritten with React Native

A step counter on Android platform. June, 2012

###DESCRIPTION:
By using the algorithm (a = sqr(x^2+y^2+z^2)), with threshold h = 1.0. Accuracy rate >= 95%.
